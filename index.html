Welcome to OSHConnect-Python’s documentation!

OSHConnect-Python

OSHConnect-Python is the Python version of the OSHConnect family of application libraries inteded to provide a simple and straightforward way to interact with OpenSensorHub (or another CSAPI server) by way of OGC API - Connected Systems. It supports or will support at the time of a 1.0 release Part 1 and Part 2 of the Connected Systems api, as well as certain streaming features made possible by OpenSensorHub.
